# Fuzz

Fuzz is a game where you write a Memory. Dissolving slowly turns it into Fuzz. You fight back with Rewriting to leave good Clues and Fresh Clues through the Endless Fight. Using its Training, the computer makes a good Best Guess during Reconstructing and shows the Reconstructed Memory as the Real Experience with the Quiet Rewrite at the end. It all feels like the Sand Drawing with The Waves coming again and again at different Fuzz Levels. The player learns the Feeling Lesson instead of making an Exact Copy by using Creative Guessing with the Smart Robot through the Word Lesson. The Rewriting gave Perfect Help. It was all the Feeling Way. It was teaching the Fixing Science through the Watch and Fight Way. The whole game was the Feeling Science.

## Language

**Memory**:

The private text submitted by the player at the beginning of a round. It is the sole subject of dissolution and reconstruction.

_Avoid_: sentence, confession, secret, thought, input, recollection

**Fuzz**:

The noisy, partially-dissolved state of a Memory. It is the visible, degraded version that the player must actively fight against in real time.

_Avoid_: noise, mess, garbage, scrambled text, random symbols, distortion

**Rewriting**:

The player typing parts of the Memory again while the Fuzz is eating it away. This is how the player tries to save the Memory.

_Avoid_: re-typing, fighting, resistance, maintenance, defending

**Dissolving**:

The slow way the Fuzz takes over more and more of the Memory by itself. Tiny waves wash away real letters and throw on random junk.

_Avoid_: forward diffusion, adding noise, destruction, erosion, washing, melting, fuzzing, the waves

**Reconstructing**:

When the computer looks at the last bits of the Memory left in the Fuzz and tries to build a clean, full version of the Memory again.

_Avoid_: reverse diffusion, fixing, rebuilding, cleaning up, the magic trick, guessing

**Reconstructed Memory**:

The clean version of the Memory that the computer shows you at the end after Reconstructing. It is almost the same as the one you wrote, but not exactly.

_Avoid_: final memory, cleaned memory, new memory, almost-memory, computer version

**Clues**:

The small pieces of the real Memory that are still left in the Fuzz. The computer uses the Clues during Reconstructing to make the Reconstructed Memory.

_Avoid_: bits, pieces, leftovers, hints, remaining letters, scraps

**Best Guess**:

The way the computer decides what the Memory was most likely to be, using the Clues. The Reconstructed Memory is the computer's Best Guess.

_Avoid_: imagination, likely version, computer idea, something close, most likely, creative guessing

**Training**:

The time before the game when the computer practiced with millions of other sentences. It learned how to turn Fuzz back into a Memory by doing the same thing over and over.

_Avoid_: practicing, learning, old games, AI training

**Fresh Clues**:

The special Clues you give the computer by Rewriting while the Fuzz is still happening, at the exact right time. These Fresh Clues help the computer make a much better Best Guess during Reconstructing.

_Avoid_: right-time clues, real clues, in-the-fight clues, live clues, timely hints

**Endless Fight**:

The feeling that you have to keep Rewriting the whole time. The Fuzz keeps coming and you never get to save the Memory once and stop.

_Avoid_: never-ending rewriting, you always have to keep going, no saving it once, endless battle

**Real Experience**:

The special way the game teaches you by making you live through the destroying, the fighting, and the fixing yourself. You don't just hear about it — you actually feel it happen.

_Avoid_: feeling the science, understanding by fighting, living the fuzz, the lesson you live, hands-on learning

**Quiet Rewrite**:

When the computer changes parts of your Memory in the Reconstructed Memory. It feels like the computer quietly rewrote what you wrote.

_Avoid_: creepy cool feeling, memory change, robot changed it, strange feeling

**Sand Drawing**:

The picture or thing you're trying to protect that keeps getting washed away by the waves. The game feels like watching a sand drawing get washed away while you desperately redraw it, and then a robot guesses what the whole picture was from the last bits you saved.

_Avoid_: washed away picture, redrawing the sand, robot's guess picture, sandcastle, sand art

**The Waves**:

The tiny waves that come during Dissolving. Each wave washes away some real letters from the Memory and throws on extra junk at the same time.

_Avoid_: washing waves, fuzz waves, tiny waves, sand waves

**Fuzz Levels**:

The different amounts of Fuzz as The Waves keep coming. At the start the Memory is only a little Fuzzed, and later it gets more and more Fuzzed.

_Avoid_: how fuzzed it is, fuzz stages, wave steps, noise levels

**Cleaning Steps**:

The small steps the computer takes during Reconstructing to turn the Fuzz back into a clean Memory. It does the fixing one little step at a time, starting from when the Memory is almost all Fuzzed and slowly making it cleaner and cleaner.

_Avoid_: fixing steps, rebuilding steps, slow clean-up

**Feeling Lesson**:

The special way the game helps you understand by making you live through the destroying, the fighting, and the fixing yourself. You feel the science instead of just hearing about it.

_Avoid_: feeling the science, the real lesson, making it real, understanding by doing

**Exact Copy**:

Trying to remember the Memory perfectly, like a photo. The computer is not doing this when it makes the Best Guess.

_Avoid_: the photocopy, remembering perfectly, the photo memory, copying exactly

**Creative Guessing**:

When the computer uses its imagination from Training plus the Clues to make the Best Guess. It is not making an Exact Copy — it is creating the most likely version.

_Avoid_: the robot's imagination, smart making up, the likely version

**Smart Robot**:

The part of the computer that uses Creative Guessing at the end to look at the last bits you saved and make the Best Guess for the Reconstructed Memory.

_Avoid_: the guessing robot, the final guesser, the robot friend

**Word Lesson**:

Learning the science by feeling it with words instead of pictures. The game uses a Memory (words) so you can go through the destroying and fixing yourself.

_Avoid_: text instead of pictures, feeling it with words, the secret science game

**Perfect Help**:

The special help that comes from giving clues at the exact right time. Your Rewriting gives the computer Fresh Clues at the exact right Fuzz Level, which helps the Smart Robot make a much better Best Guess later.

_Avoid_: the right moment clues, super strong clues, the timing power

**Feeling Way**:

The special way the game helps you understand by going through it all yourself. You watch the destroying, fight it, see the fixing, and feel that the fixing is Creative Guessing not an Exact Copy.

_Avoid_: the cool lesson, going through it, the experience science

**Fixing Science**:

The science the game is secretly teaching. It is the way computers can rebuild things from Fuzz using the Smart Robot and Creative Guessing.

_Avoid_: the guessing science, the destroy and fix idea, computer magic

**Watch and Fight Way**:

The way the game helps you understand by watching the destroying, fighting it yourself, and seeing the fixing. You learn by going through the whole thing.

_Avoid_: seeing it happen, doing it yourself, the live lesson

**Feeling Science**:

The big reason the game exists. It is the science of how computers can fix very messy things, and the game helps you feel it by going through the destroying, fighting, and fixing yourself.

_Avoid_: the learning game, the experience game, the science you feel
